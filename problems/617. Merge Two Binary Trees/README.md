## 题目描述

  把两棵二叉树重叠合并,每个节点相加

## 例子
```
Input: 
	Tree 1                     Tree 2                  
          1                         2                             
         / \                       / \                            
        3   2                     1   3                        
       /                           \   \                      
      5                             4   7                  
Output: 
Merged tree:
	     3
	    / \
	   4   5
	  / \   \ 
	 5   4   7
```
## 解题思路

采用递归的方式遍历两棵树,以t1树作为基准树,将t2累加到t1上
   1. 若t1树的左/右子树为null,则将t2的左/右直接挂载到t1上,并且不继续向下遍历
   2. 若t2树的左/右子树为null,则不做处理,向下遍历时传null
   3. 累加若是null则加0
   4. 对传入的t1/t2进行null判断

优化:
在进入子树后进行累加,采用返回值的方式,再对父节点的left&right进行赋值,将累加后的子树挂载到父节点上

## 结果

    104ms
