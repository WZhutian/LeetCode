## 题目描述

  给一串数组和一个目标值,求目标值是由哪2个数相加而得的

## 例子
```
Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].
```
## 解题思路
**JAVA**

目标值:x

当前遍历的数:a

使用map建立{数:索引}键值对,遍历数组,查找是否存在 x-a 这个键

**JavaScript**

目标值:x

当前遍历的数:a

遍历数组,查找数组中是否存在另一个等于 x - a 的数

## 结果

beats 59.26%
